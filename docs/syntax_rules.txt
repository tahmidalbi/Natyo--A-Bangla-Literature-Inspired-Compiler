Natyo Syntax Rules

1. Program Structure
A program starts with:
    prostabona

A program ends with:
    uposhonghar

2. Variable Declaration
Integer:
    shonkhya a = 5;

Float:
    purnoshonkhya b = 3.14;

String:
    bakkyo msg = "hello";

Char:
    okkhyor c = 'A';

3. Assignment
Examples:
    a = 10;
    msg = "Natyo";
    c = 'B';

4. Print
Use shobdo:
    shobdo a;
    shobdo msg;
    shobdo shakti(2, 3);

5. Input
Use jobdo:
    jobdo a;

6. Conditional
Use jodi and onnotha:
    jodi (a > 5)
        shobdo a;
    onnotha
        shobdo 0;

7. Loop
Use chokro:
    chokro (i = 0; i < 5; i = i + 1)
        shobdo i;

8. Function Definition
Use drissho:
    drissho jog(a, b)
        pherot a + b;

9. Function Return
Use pherot:
    pherot expression;

10. Comments
Single-line:
    $$ this is a comment

Multi-line:
    $*
    this is a multi-line comment
    *$

11. Statement End
Most simple statements end with semicolon.
