2. Identifiers
Examples:
- a
- total
- msg
- jog

Rule:
- starts with letter or underscore
- may contain letters, digits, underscore

3. Literals
Integer:
- 5
- 100

Float:
- 3.14
- 2.0

String:
- "hello"
- "Natyo compiler"

Char:
- 'A'
- 'x'

4. Operators
Arithmetic:
- +
- -
- *
- /
- %

Assignment:
- =

Relational:
- >
- <
- >=
- <=
- ==
- !=

5. Symbols
- ;
- ,
- (
- )

6. Comments
Single-line:
- $$ comment text

Multi-line:
- $* comment text *$

7. Invalid Tokens
Any unsupported character should produce lexical error.
