Project Name: Natyo
Type: A narrative-driven Bangla-inspired mini programming language

Purpose:
Natyo is a custom compiler language inspired by Bangla words and dramatic/narrative structure.
It is designed for compiler laboratory work with creative syntax and technical features.

Core Narrative Structure:
- prostabona = program start
- uposhonghar = program end
- drissho = function definition / scene

Core Data Types:
- shonkhya = integer
- purnoshonkhya = float
- bakkyo = string
- okkhyor = char

Input / Output:
- shobdo = print
- jobdo = input

Control Flow:
- jodi = if
- onnotha = else
- chokro = loop

Math Functions:
- shakti(base, power)
- borgomul(number)

Function Return:
- pherot = return

Preloaded Constants:
- PAI = 3.1416
- E = 2.718
- G = 9.8

Comments:
- Single-line comments start with $$ and continue to end of line
- Multi-line comments start with $* and end with *$

Special Feature:
- Runtime history log will later store variable changes in history_log.txt

Compiler Phases:
1. Lexical analysis using Flex
2. Syntax analysis using Bison
3. Semantic analysis using symbol table and type checks
4. Execution / interpretation
5. Runtime history logging
